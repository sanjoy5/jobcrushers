# JobCrushers

This project was created with ReactJS.

## Features

- Fully Responsive design
- Dynamic data fetching from json file
- Dynamicaly display Details Data
- Set and get data in Local Storage and dispaly data
- Added Assignment Marks in Chart using Rechart js 
- Added Blog Page 
- Filter and Load more data functionality added


## Installation

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server
